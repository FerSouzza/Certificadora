# Descrição do Projeto

<p>O <b>C-Learning Quest</b> é um Recurso Educacional Aberto (REA) desenvolvido para a disciplina de Certificadora de Competência Comum. O sistema consiste em uma página web interativa que utiliza um jogo no estilo arcade para ensinar a sintaxe e a lógica da <b>Linguagem C</b> de maneira acessível e lúdica.</p>

<p>A plataforma é estruturada em uma <i>Single Page Application</i> (SPA), onde o usuário navega por "Code Gates" (Portões de Código). Para progredir no jogo, o estudante deve resolver desafios de programação em tempo real. Cada fase é acompanhada por um <b>Manual de Instruções</b> composto por vídeos animados desenvolvidos no <b>VideoScribe</b>, que explicam conceitos como variáveis, estruturas de decisão e laços de repetição.</p>

<p>O público-alvo são estudantes iniciantes em tecnologia e lógica de programação, proporcionando um ambiente onde a teoria dos algoritmos é aplicada imediatamente na prática do jogo.</p>
