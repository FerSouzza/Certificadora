# Videoaulas de Linguagem C
Este projeto reúne videoaulas voltadas ao ensino da linguagem de programação C, com foco em iniciantes e estudantes que desejam fortalecer sua base em lógica e desenvolvimento.

## Conteúdo abordado
- Variáveis e Tipo de Dados
- Estruturas de decisão
- Estruturas de repetição
- Funções
